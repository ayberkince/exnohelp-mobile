# Client Funnel v1

## Purpose
This document defines the client journey funnel.

---

## Core Client Funnel
1. signup_completed
2. role_selected = client
3. onboarding_completed
4. support_request_published OR helper_search_opened
5. booking_started
6. payment_succeeded
7. booking_confirmed
8. booking_completed
9. review_submitted
10. repeat_booking_started

---

## Key Conversion Questions
- What percentage of signups choose client role?
- What percentage complete onboarding?
- What percentage publish a request or browse helpers?
- What percentage start a booking?
- What percentage complete payment?
- What percentage complete a booking?
- What percentage return for a second booking?

---

## Important Drop-Off Points
- signup -> onboarding complete
- onboarding complete -> first request / first browse
- booking started -> payment success
- booking confirmed -> booking completed

# Helper Funnel v1

## Purpose
This document defines the helper journey funnel.

---

## Core Helper Funnel
1. signup_completed
2. role_selected = helper
3. helper_profile_completed
4. verification_submitted
5. verification_approved
6. requests_feed_opened
7. request_application_submitted
8. booking_confirmed
9. booking_completed
10. payout_released
11. second application or second booking

---

## Key Conversion Questions
- What percentage of signups choose helper role?
- What percentage complete profile?
- What percentage submit verification?
- What percentage get approved?
- What percentage browse requests?
- What percentage apply?
- What percentage complete a first booking?
- What percentage return for another booking?

---

## Important Drop-Off Points
- signup -> profile complete
- profile complete -> verification submitted
- verification submitted -> approved
- approved -> first request application
- first application -> first completed booking

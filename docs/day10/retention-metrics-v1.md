# Retention Metrics v1

## Purpose
This document defines how retention is measured for both sides of the marketplace.

---

## 1. Client Retention
Important retention signals:
- returned within 7 days
- returned within 30 days
- created another request
- completed another booking
- reused same helper
- left another review

### Core retention metric
Repeat booking rate within 30/60/90 days

---

## 2. Helper Retention
Important retention signals:
- returned to requests feed
- applied to another request
- completed another booking
- remained active and available
- maintained verification and profile quality

### Core retention metric
Percentage of approved helpers who complete another booking within 30/60/90 days

---

## 3. Session-Level Retention
Useful indicators:
- weekly active clients
- weekly active helpers
- monthly active clients
- monthly active helpers

---

## 4. Retention Rule
For this marketplace, retention should be measured by meaningful actions, not just app opens.

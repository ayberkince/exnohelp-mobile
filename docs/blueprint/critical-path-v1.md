# Critical Path v1
## The Non-Negotiable Build Sequence
1. **Auth:** Control the gates.
2. **Onboarding:** Ensure profile quality.
3. **Discovery:** Allow Clients to find Helpers.
4. **Booking:** Enable the transaction.
5. **Payments:** Make the money real.
6. **Safety:** Build the reporting/moderation layer.
7. **Beta Ops:** Launch the waitlist and dashboard.

## Rule
Any work not on this path is deferred until Stage 4 (Post-Launch).

# Master Startup Blueprint v1

## Definition
A trust-first two-sided marketplace connecting people needing non-medical support around medical processes with verified helpers in Berlin.

## Core Modules
- Auth & Roles (Admin/Helper/Client)
- Onboarding & Profile Verification
- Request Marketplace & Helper Discovery
- Booking State Machine (Pending -> Completed)
- Payments (Stripe) & Payout Logic
- Messaging & Trust/Safety Tools
- Public Site & Waitlist

## Business Rules
1. Trust beats Growth.
2. Core Flow beats Polish.
3. Liquidity beats Breadth.

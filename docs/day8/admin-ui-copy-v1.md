## Reports Queue
Headline:
Open reports

Body:
Review reports, assess severity, and take documented action where necessary.

Buttons:

- Mark Under Review
- Close Report

Headline:

Headline:

This will block the user from normal platform access until further review.

Buttons:
- Confirm Suspension
- Cancel

---

## Refund Review
Headline:
Review refund eligibility

Body:
Check booking, payment, and report details before approving or denying refund action.

# Role Permissions v1

## Client Permissions
A client can:
- create and edit their profile
- create support requests
- browse helpers
- view helper profiles
- submit bookings
- pay for bookings
- send messages in active conversations
- leave reviews after completion
- report helpers, messages, or bookings
- cancel under allowed policy conditions

A client cannot:
- access admin tools
- verify helpers
- alter payout records
- change booking records outside allowed actions
- bypass platform payment rules

---

## Helper Permissions
A helper can:
- create and edit their profile
- submit verification information
- set availability
- browse open requests
- apply to requests
- accept eligible bookings
- send messages in active conversations
- view their earnings summary
- report clients, messages, or bookings
- pause their profile availability

A helper cannot:
- act as admin
- approve their own verification
- alter payout calculations
- provide prohibited medical services through the platform
- bypass platform payment rules

---

## Admin Permissions
An admin can:
- review helper verification submissions
- review reports
- suspend or reactivate users
- monitor bookings
- issue refunds or support refund workflows
- view moderation records
- update report states
- mark payout holds where necessary
- record admin actions

An admin cannot:
- act outside documented moderation policy
- silently alter user financial records without logged action

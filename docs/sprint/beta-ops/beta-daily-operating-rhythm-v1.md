# Beta Daily Operating Rhythm v1
## Morning Block (08:00 - 10:00)
- Check system health/logs (Vercel/Sentry).
- Review Support inbox & Report queue.
- Inspect active/pending bookings.
## Midday Block (12:00 - 14:00)
- Respond to users.
- Manually unblock stuck onboardings.
- Review verification queue.
## Evening Block (18:00 - 20:00)
- Review behavior (Posthog/Analytics).
- Classify issues (Urgent vs. Later).
- Set priorities for tomorrow.

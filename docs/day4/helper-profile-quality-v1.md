# Helper Profile Quality v1

## Purpose
This document defines what a good helper profile looks like.

Helper profile quality matters because trust is central to the platform.

---

## A strong helper profile should include:
- clear profile photo
- real first name
- warm but concise bio
- visible languages
- visible city/district
- visible service categories
- hourly rate
- availability
- verification status
- trust-oriented layout

---

## Minimum quality requirements
A helper profile should not be publicly active unless it includes:
- profile photo
- first name
- city/district
- at least one language
- short bio
- at least one support category
- hourly rate
- availability

---

## Bio guidance
The bio should feel:
- human
- reassuring
- simple
- respectful
- trustworthy

Good bio qualities:
- calm tone
- short and clear
- non-medical positioning
- respectful language

Bad bio qualities:
- exaggerated claims
- medical claims
- vague or empty wording
- overly casual or inappropriate tone

---

## Trust markers on profile
Important trust elements:
- verification badge
- number of completed bookings
- rating
- response rate
- clear service categories
- platform safety note

---

## Marketplace rule
Low-quality profiles reduce trust and should be improved before full marketplace exposure.

# Value Proposition v1

## Purpose
This document defines the value proposition for each side of the marketplace.

---

## 1. Value Proposition for Clients

### Core message
Get trusted non-medical support when you do not want to face an appointment or recovery moment alone.

### Functional value
- easier appointment support
- less stress
- practical help
- companionship
- simple booking flow
- structured trust signals

### Emotional value
- calmer experience
- less isolation
- more confidence
- more reassurance

---

## 2. Value Proposition for Helpers

### Core message
Earn flexible side income by providing meaningful, structured support to people who need it.

### Functional value
- flexible opportunities
- clear service categories
- in-platform booking and payment
- profile-based visibility
- trust-building through verification and reviews

### Emotional value
- meaningful work
- sense of usefulness
- respectful service role

---

## 3. Value Proposition for the Marketplace as a Whole
The platform fills the gap between formal medical care and informal human support.

It makes that gap:
- easier to access
- safer to organize
- clearer to understand
- more trusted than random informal arrangements

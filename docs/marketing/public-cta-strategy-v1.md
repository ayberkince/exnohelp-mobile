# Public CTA Strategy v1

## Purpose
This document defines what calls-to-action appear on the public website.

---

## Primary CTAs
- Get Support
- Become a Helper

These should appear:
- in hero
- in final section
- in role-specific page sections

---

## Secondary CTAs
- Learn How It Works
- Read Trust & Safety
- View FAQs

These reduce friction for uncertain users.

---

## CTA Principles
- always make role paths obvious
- never force too many choices at once
- trust information should be easy to access before signup
- every page should have one clear next step
